Basics

Core concepts of Python including variables, data types, operators, control flow, loops, and basic syntax. These form the foundation for any Python program.

Introduction & Syntax

Python was created by Guido van Rossum and first released in 1991. It's known for its clean, readable syntax and "batteries included" philosophy with a rich standard library.

Python uses indentation for blocks instead of braces. It's dynamically typed and supports multiple paradigms.

  • Introduction: Easy-to-read, versatile language used for web development, data science, automation, AI, and more.
  • Syntax: No semicolons required; indentation defines code blocks. Comments use #.
  • Output: Use print() function.
# Simple Hello World
print("Hello, World!")  # Outputs: Hello, World!

# Formatted output
name = "Python"
version = 3.12
print(f"Welcome to {name} {version}")  # Outputs: Welcome to Python 3.12

Variables & Data Types

Variables are created by assignment. Common types: int, float, str, bool, list, tuple, dict, set.

  • Variables: No declaration needed; dynamically typed.
  • Data Types: Numbers (int, float, complex), sequences (str, list, tuple), mappings (dict), sets.
x = 5          # int
y = 3.14       # float
name = "Python" # str
is_fun = True  # bool

# Lists (mutable) and Tuples (immutable)
fruits = ["apple", "banana", "cherry"]  # list
coords = (10, 20)                       # tuple

# Dictionary (key-value)
person = {"name": "Alice", "age": 30}

# Type checking and conversion
print(type(x))      # 
z = float(x)        # 5.0

Control Flow & Loops

Use if-elif-else for decisions, for/while for loops, and comprehensions for concise iteration.

  • Conditionals: if, elif, else.
  • Loops: for (iterates over sequences), while (condition-based).
# If-elif-else
age = 18
if age >= 18:
    print("Adult")
elif age >= 13:
    print("Teen")
else:
    print("Child")

# For loop
for fruit in fruits:
    print(fruit)

# Range-based for loop
for i in range(5):
    print(i)  # 0 to 4

# While loop
count = 0
while count < 3:
    print(count)
    count += 1

# List comprehension
squares = [x**2 for x in range(10)]

Strings & Lists

Strings are immutable sequences with many methods. Lists are mutable and versatile.

# String operations
greeting = "Hello Python"
print(len(greeting))          # 12
print(greeting.upper())       # HELLO PYTHON
print(greeting[0:5])          # Hello (slicing)

# List operations
numbers = [1, 2, 3]
numbers.append(4)             # [1, 2, 3, 4]
numbers.pop()                 # removes last
print(sorted(numbers))        # [1, 2, 3]

Functions

Functions are defined with def, support parameters, defaults, *args, **kwargs, and lambdas.

Functions & Parameters
def greet(name="World"):
    print(f"Hello, {name}!")

greet()           # Hello, World!
greet("Alice")    # Hello, Alice!

# Variable arguments
def sum_all(*args):
    return sum(args)

print(sum_all(1, 2, 3, 4))  # 10

# Keyword arguments
def info(**kwargs):
    for key, value in kwargs.items():
        print(f"{key}: {value}")

info(name="Bob", age=25)

Advanced Functions
# Recursion
def factorial(n):
    if n <= 1:
        return 1
    return n * factorial(n-1)

# Lambda
square = lambda x: x**2
print(square(5))  # 25

Object-Oriented Programming

Python supports classes, inheritance, polymorphism, encapsulation, and more.

Classes & Objects
class Car:
    def __init__(self, model, year):
        self.model = model
        self.year = year
    
    def drive(self):
        print(f"Driving {self.model}")

my_car = Car("Tesla", 2023)
my_car.drive()
Inheritance & Polymorphism
class ElectricCar(Car):
    def drive(self):
        print(f"Driving electrically {self.model}")

vehicle = ElectricCar("Tesla", 2023)
vehicle.drive()  # Polymorphism

Advanced Topics

Modules, exceptions, file I/O, comprehensions, decorators, generators, and more.

Modules & Exceptions
import math
print(math.sqrt(16))  # 4.0

try:
    x = 10 / 0
except ZeroDivisionError:
    print("Cannot divide by zero")
File Handling & Comprehensions
# Writing/Reading files
with open("example.txt", "w") as f:
    f.write("Hello Python")

with open("example.txt", "r") as f:
    print(f.read())

# Dictionary comprehension
squares = {x: x**2 for x in range(5)}
