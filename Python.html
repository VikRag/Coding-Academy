Basics

Start with variables, types, loops, and simple code.

Hello World & Syntax

Python uses spaces for blocks. No {} or ; needed. Use # for comments.

# Hello World
print("Hello, World!")

# With name
name = "Python"
print(f"Hi {name}")
Variables & Types

No need to declare type. Common: int, float, str, bool, list.

x = 5          # int
y = 3.14       # float
name = "Hi"    # str
active = True  # bool

fruits = ["apple", "banana"]  # list

print(type(x))  #
If & Loops

Use if for decisions. for/while for repeats.

# If
age = 18
if age >= 18:
    print("Adult")
else:
    print("Kid")

# For loop
for fruit in fruits:
    print(fruit)

# While
count = 0
while count < 3:
    print(count)
    count += 1
Strings & Lists

Strings can't change. Lists can.

text = "Hello"
print(text.upper())  # HELLO
print(text[0])       # H

nums = [1, 2]
nums.append(3)       # [1,2,3]
print(nums)

Functions

Reuse code with def.

Simple Functions
def greet(name="World"):
    print(f"Hello {name}")

greet()       # Hello World
greet("Ali")  # Hello Ali

def add(a, b):
    return a + b

print(add(2, 3))  # 5
Lambda & Recursion
# Lambda (short function)
sq = lambda x: x * x
print(sq(4))  # 16

# Recursion
def fact(n):
    if n <= 1:
        return 1
    return n * fact(n-1)

print(fact(5))  # 120

OOP (Classes)

Group data and actions.

Class & Object
class Dog:
    def __init__(self, name):
        self.name = name
    
    def bark(self):
        print("Woof!")

my_dog = Dog("Buddy")
my_dog.bark()
Inheritance
class Puppy(Dog):
    def bark(self):
        print("Yip!")

small = Puppy("Tiny")
small.bark()  # Yip!
