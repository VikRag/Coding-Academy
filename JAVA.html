Java Programming Language

Java is a high-level, object-oriented, general-purpose programming language that runs on the Java Virtual Machine (JVM). Learn key topics and practice writing code in this interactive-style page.

Topics to Learn in Java

  • Java Basics: Variables, Data Types, Operators
  • Control Flow: if, else, switch, loops
  • Object-Oriented Concepts: Classes, Objects, Inheritance, Polymorphism
  • Collections & Data Structures: Arrays, ArrayList, HashMap
  • Exception Handling: try, catch, finally
  • Input/Output: Scanner, File I/O
  • Multithreading & Concurrency
  • Java Libraries & APIs
  • GUI Programming: Swing / JavaFX

About Java

Java is platform-independent, object-oriented, secure, and robust. It is widely used in mobile applications (Android), enterprise software, web servers, and cloud services. Java programs run on the Java Virtual Machine (JVM) ensuring portability across platforms.

  • Paradigm: Object-Oriented, Concurrent, Class-Based
  • Platform: JVM (Write Once, Run Anywhere)
  • Strengths: Security, Stability, Large Libraries, Community Support
// Example: Simple Java Program
public class HelloWorld {
    public static void main(String[] args) {
        // Print text to console
        System.out.println("Hello, World!");
        
        // Variables
        int age = 18;
        double price = 19.99;
        
        // Conditional
        if(age >= 18) {
            System.out.println("Adult");
        } else {
            System.out.println("Minor");
        }
    }
}

Java Usage & History

Java was released in 1995 by Sun Microsystems and later acquired by Oracle. Java continues to be one of the most widely-used programming languages in the world, powering:

  • Android mobile apps
  • Enterprise and server applications
  • Web backend systems
  • Games and GUI applications
  • Cloud services and large-scale applications
